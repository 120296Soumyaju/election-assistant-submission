# VoterVault AI | Secure Election Guide

**VoterVault AI** is a high-fidelity, secure voter education platform built for the Google for Developers Virtual Prompt Wars (Challenge 2). It combines the intelligence of **Google Gemini** with the security of **Web3 (MetaMask)** to provide a trusted, interactive journey through the electoral process.

## Unique Selling Propositions (USPs)

- **Cryptographic Vault (Web3):** The first election assistant to use MetaMask-based encryption (AES-256) for API key security. Only the wallet owner can unlock the assistant.
- **Visual Timeline Integration:** A high-end interactive timeline that dynamically updates the AI's context as you learn.
- **Premium Glassmorphism Design:** A "Cyber-Patriot" aesthetic with animated gradient meshes, fluid transitions, and the modern 'Outfit' typeface.
- **Zero-Knowledge Architecture:** No backend, no database, no server. 100% private and static.

## Tech Stack

- **Core:** React, Vite, Vanilla CSS.
- **AI:** @google/genai (Gemini Pro/Flash).
- **Web3:** Ethers.js, MetaMask.
- **Security:** Crypto-JS (AES-256).

### Setup Instructions
1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev`.
4. **Connect MetaMask** in the top right corner.
5. **Paste your Gemini API Key** and click **"Secure Vault"**.
6. Sign the message in MetaMask (Free/No Gas) to encrypt your key.
7. To use the app later, simply click **"Unlock Vault"** and sign again!
